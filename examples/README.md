# Tag-Press Beispielseiten

Dieses Verzeichnis enthält vollständig validierte Beispielseiten als Referenz für Azubis.

## Übersicht

| Datei | Beschreibung |
|-------|--------------|
| `page-a-startseite.md` | Detaillierte Analyse der Startseite |
| `page-b-about.md` | Beispiel "Über uns" Seite |
| `page-c-contact.md` | Beispiel Kontaktseite |
| `invalid-examples.md` | Absichtlich fehlerhafte Beispiele |

## Wie man diese Beispiele nutzt

1. **Lesen und verstehen** – Jedes Beispiel erklärt die Designentscheidungen
2. **Nachbauen** – Versuche die Beispiele selbst zu implementieren
3. **Validieren** – Prüfe ob deine Implementation die gleichen Ergebnisse liefert
4. **Erweitern** – Füge eigene Zonen oder Objekte hinzu

## Wichtige Lernpunkte

- Warum wurden bestimmte Zonen gewählt?
- Wie hängen Objekte semantisch zusammen?
- Welche Constraints gelten?
- Wie sieht gültiger vs. ungültiger Code aus?
